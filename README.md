# 🧠 Mind Tracker

Mind Tracker is a full-stack MERN web application that helps users track and analyze their mental well-being. It allows users to monitor mood, stress levels, and personality through interactive questionnaires and provides instant results.

---

## 🚀 Features

- 📊 Mood Tracking  
- 😟 Stress Level Analysis  
- 🧩 Personality Checker  
- ⚡ Real-time Result Generation  
- 💾 Data Storage using MongoDB  
- 🎯 Simple and User-Friendly Interface  

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

git clone https://github.com/your-username/mind-tracker.git


### 2️⃣ Install dependencies

#### Frontend

cd frontend
npm install
npm run dev


#### Backend

cd backend
npm install
node server.js


---

## ▶️ Usage

1. Open the application in browser  
2. Select a module (Mood / Stress / Personality)  
3. Answer the questions  
4. View the result instantly
